# logout

Logout and clear stored credentials.

## Usage

```bash
willhaben logout
```

## Example

```bash
willhaben logout
# Logged out successfully
```

## What Happens

1. Local tokens are removed from `~/.willhaben/config.json`
2. You'll need to login again to access your listings

## If Not Logged In

```bash
willhaben logout
# Not logged in
```
